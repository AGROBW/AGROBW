🚀 COMO INICIAR:
─────────────────────────────────────────────────────────────────────────────

PASSO 1: Executar Migração SQL (OBRIGATÓRIO)
──────────────────────────────────────────────
1. Vá para https://app.supabase.com/
2. Selecione seu projeto BWAGRO
3. SQL Editor → New Query
4. Copie o conteúdo de: supabase-migrations-address.sql
5. Clique em ▶️ RUN


PASSO 2: Testar Localmente (RECOMENDADO)
──────────────────────────────────────────
$ npm run dev
→ Acesso: http://localhost:3001/register
→ Teste CEP: 01310-100 (São Paulo)
→ Verifique auto-preenchimento


PASSO 3: Verificar no Supabase (CONFIRMAÇÃO)
──────────────────────────────────────────────
1. Supabase Dashboard
2. Tables → users
3. Procure o novo usuário
4. Verifique campos preenchidos


PASSO 4: Deploy em Produção (QUANDO PRONTO)
──────────────────────────────────────────────
$ git add -A
$ git commit -m "feat: expandir formulário de cadastro"
$ git push origin main
→ Vercel faz deploy automático
